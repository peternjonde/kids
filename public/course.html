With time we plan on teaching the kids the 3 basic programing languages

Python

Python is the 3rd most used programming language in 2023. Unlike other languages python has much easier syntax thus making it the ideal beginner's language.

Java

Java is the 6th most used programming language in 2023. Java may be less used now than other languages, but it has very strict syntax and gives the programmer more control of the code by controlling memory etc. Java is great at just developing how to think like a programmer and will make other languages feel easy.

C++

C++ is the 4th most used programming language in 2023. C++ is an easier version of the language C. C++ is like the half-way between python and java in terms of complexity. Working with C++ is a great way to show how well you understand code.
